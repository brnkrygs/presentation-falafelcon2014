Talks

ASP .NET vNext

by doing...

  • Drop system.web (move to nuget)
  • Move to github
  • Faster dev cycle
  • Use any editor (no .csproj!)
  • Host wherever with OWIN (express via IAppBuilder)
  • Faster - less build artifacts, keep build in memory
